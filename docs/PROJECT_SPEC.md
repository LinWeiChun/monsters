# 貘nsters 專案需求規格

## 一、專案概述

貘nsters 是一套以「食夢貘吃掉負面情緒」為概念的情緒記錄與紓壓系統。使用者可以透過文字、錄音、照片、影片與畫心情等方式記錄煩惱或日記，並透過怪獸聊天室、心的軌跡、社群、圖鑑與互動區獲得陪伴感與紓壓體驗。

本版本目標支援：

- Android App
- iOS App
- Web 瀏覽器

## 二、核心使用者

1. 有煩惱但無從宣洩，或想記錄生活、煩惱與心情的使用者。
2. 想透過互動區進行紓壓、測驗、遊戲或心理狀態探索的使用者。
3. 喜歡簡單直覺介面、可愛怪獸與蒐集體驗的使用者。

## 三、核心功能模組

### 3.1 會員與登入

- 使用者註冊
- 使用者登入
- Google 登入
- 首次 Google 登入設定個人資料
- 忘記密碼
- 登出

### 3.2 個人資料

- 查詢個人資料
- 修改個人資料
- 更改個人頭貼
- 設定或更改四位數密碼鎖
- 忘記密碼鎖

### 3.3 煩惱功能

使用者可以新增、查詢、修改與分享煩惱。

新增煩惱流程：

1. 進入怪獸聊天室
2. 選擇煩惱類別
3. 選擇記錄方式
4. 輸入煩惱內容
5. 畫心情
6. 記錄煩惱分數
7. 選擇是否分享
8. 獲得隨機怪獸或獎勵

### 3.4 日記功能

使用者可以新增、查詢、修改與分享日記。

新增日記流程：

1. 進入日記聊天室
2. 選擇記錄方式
3. 輸入日記內容
4. 畫心情
5. 記錄心情分數
6. 選擇是否分享
7. 獲得隨機怪獸或獎勵

### 3.5 歷史記錄

- 查看過去新增的煩惱與日記
- 修改煩惱狀態
- 將未解決煩惱更新為已解決
- 分享或取消分享煩惱
- 分享或取消分享日記

### 3.6 心的軌跡

- 統整煩惱與日記中的情緒分數
- 以圖表呈現心情變化
- 最多顯示最近七次情緒變化

### 3.7 怪獸圖鑑

- 查看所有怪獸
- 查看已獲得怪獸
- 新增煩惱或日記後隨機獲得怪獸
- 更換怪獸造型
- 每日測驗累積獎勵可獲得怪獸造型

### 3.8 社群

- 查看已分享的煩惱與日記
- 對社群文章按愛心
- 對社群文章留言
- 使用匿名或保護隱私方式呈現內容

### 3.9 互動區

互動區包含：

- 解答之書
- 每日測驗
- 深度心理測驗
- 心理小遊戲
- 紓壓方法

### 3.10 使用說明與回饋

- 查看使用說明
- 新增使用回饋
- 分享貘nsters App

## 四、非功能需求

### 4.1 可用性

- 介面需簡單直覺。
- 手機版以單手操作為優先。
- Web 版需維持基本可讀性與可操作性。

### 4.2 安全性

- 個人資料需受保護。
- 煩惱與日記預設為私人資料。
- 分享狀態必須由使用者控制。
- 密碼與密碼鎖不得明文保存。

### 4.3 可維護性

- 前後端分離。
- 模組化開發。
- API 與資料結構需文件化。
- 不得將商業邏輯混入 UI 或 Controller。

### 4.4 跨平台

- Flutter 專案需支援 Android、iOS、Web。
- Web 版若有平台限制，需提供替代方案或明確說明。

## 五、開發優先順序

1. 專案基礎架構
2. 會員登入與個人資料
3. 煩惱功能
4. 日記功能
5. 歷史記錄與心的軌跡
6. 怪獸圖鑑
7. 社群
8. 互動區
9. 使用說明、回饋、分享 App
10. 測試、優化與部署
